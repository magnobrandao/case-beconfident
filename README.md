# Case Técnico BeConfident - Simulação de Plataforma

Este projeto trata-se de uma **simulação** desenvolvida para o Case Técnico da **BeConfident**. 

O seu propósito é servir como um **esboço de plataforma**, criando um ambiente realista e funcional para facilitar **futuras análises** operacionais, diagnósticos de performance de mídia paga e tomada de decisão estratégica.

## Sobre a Aplicação

- **Finalidade:** Fornecer um ambiente de análise de performance de tráfego pago com dados cruzados.
- **Escopo atual:** O projeto exibe dados simulados (mockados) cruzando métricas de plataformas como **Meta Ads**, **Google Search**, **YouTube** e **Demand Gen** diretamente com o faturamento final no Checkout.
- **Próximos Passos:** O esboço foi criado prevendo integrações reais e funcionalidades mais complexas de diagnóstico (como "Falso Campeão", "Hemorragia Infantil", etc.).

## Tecnologias Utilizadas

- **Frontend:** React, TypeScript, Vite
- **Estilização:** TailwindCSS / CSS Vanilla
- **Visualização:** Dashboards dinâmicos com análise de KPIs (ROAS, CPA, Investimento).

## Como executar localmente

1. Entre no diretório do front-end:
   ```bash
   cd web
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
